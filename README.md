# 🎲 Shut the Box (Web Oyunu / Web Game)

---

## 🇹🇷 Türkçe

Modern ve tarayıcı tabanlı **Shut the Box** oyunu. HTML, CSS ve JavaScript ile geliştirilmiştir.

### 🚀 Özellikler
- Tek oyunculu ve çok oyunculu modlar
- Sırayla ve karşılıklı oynama
- Dinamik zar sistemi
- Geçerli hamle kontrolü
- Skor ve tur sistemi
- Mobil uyumlu modern arayüz

### 🕹️ Nasıl Oynanır?
- Zar atılır  
- Zar toplamına eşit taşlar seçilir  
- Amaç tüm taşları kapatmaktır  
- Hamle kalmazsa oyun biter  

### 📁 Dosya Yapısı
- `index.html` → Arayüz  
- `style.css` → Tasarım  
- `script.js` → Oyun mantığı  

---

## 🇬🇧 English

A modern browser-based **Shut the Box** game built with HTML, CSS, and JavaScript.

### 🚀 Features
- Singleplayer and Multiplayer modes  
- Turn-based and versus gameplay  
- Dynamic dice system  
- Valid move checking  
- Score and round system  
- Responsive modern UI  

### 🕹️ How to Play
- Roll the dice  
- Select tiles matching the dice total  
- Goal is to close all tiles  
- Game ends when no moves remain  

### 📁 Project Structure
- `index.html` → UI  
- `style.css` → Styling  
- `script.js` → Game logic
